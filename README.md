# Maglor

dmenu integration for bw-cli

# Requirements

- node.js
- npm
- make
- bitwarden cli
- dmenu
- xclip

# Installation

```Bash
make build
```

```Bash
sudo make install
```

# Usage

Run the local bitwarden service. You must be logged in with the bitwarden cli.

```Bash
bw serve --port 8080
```

run

```Bash
maglor
```
